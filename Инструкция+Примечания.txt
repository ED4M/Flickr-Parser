Готовый билд проекта в папке /build, исходные файлы в /src.
Проект был создан с ипользованием create-react-app.

Примечания: 

1) В задании было указано выбрать список популярных постов, но Flickr не обладает нативным методом получения популярных постов.
Я решил выбрать самый популярный тег за неделю и вывести по нему посты по "интересности" (как написано в документации к API Flickr).
Надеюсь, этот момент будет не критичен :)
Полную модель работы с API можно увидеть в файле "flickr-service.js".

2) При нажатии на сердце, alert выскакивает с идентификатором фото с сервера.
